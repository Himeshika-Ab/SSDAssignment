Provide authentication for Github using OAuth2.0


1.  Download the repository to your local machine.
2.  Open the command line inside the SSDAssignment folder.
4.  Install all the node modules.
        npm install
5.  To start the server, run the following command in the terminal.
        node index.js
6.  Open the browser and go to http://localhost:4000
7.  By clicking on the login button you can login to the application.
